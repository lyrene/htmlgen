WER2001 - 4th Workshop on Requirements Engineering, Buenos Aires - Argentina

Reutilização de Objetivos na Engenharia de Requisitos

Karla Donato Fook; Zair Abdelouahab

PDF Scholar

Abstract

Este artigo apresenta um método de reutilização de requisitos através os objetivos, chamado MROBJER. O método permite a reutilização de requisitos de um sistema através a reutilização de seus objetivos. A reutilização de objetivos pode ocorrer entre aplicações pertencentes ao mesmo domínio denominada reutilização interdominio e também entre aplicações de diferentes domínios denominada reutilização intersistemas. MROBJER prove também o reuso parametrizado de objetivos. Isto é, os objetivos genéricos possam ser reutilizados por outras aplicações passando-se como parâmetros fatores diferenciadores dentro dos objetivos. Para auxiliar o desenvolvedor, uma ferramenta foi desenvolvida, chamada FROBJER. Um exemplo na área bancaria é ilustrado para mostrar o uso do método.
